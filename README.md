# Kamat Bakery — Official Website

A clean, responsive static website for **Kamat Bakery**, Dharwad's beloved neighbourhood bakery. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Live Site

> Deploy via [Netlify Drop](https://app.netlify.com/drop) or GitHub Pages for a live URL.

## Features

- Responsive design — works on mobile, tablet, and desktop
- Smooth scroll-reveal animations
- Interactive menu with category filters (Breads / Cakes / Snacks)
- Embedded Google Maps with the bakery's exact location
- Warm, bakery-themed colour palette

## Sections

| Section | Description |
|---|---|
| Hero | Welcome banner with key stats |
| About | Bakery story and highlights |
| Menu | Product cards with category filter tabs |
| Specialties | Key differentiators |
| Reviews | Customer testimonials |
| Contact | Address, hours, and embedded map |

## Getting Started

No build step required. Just open `index.html` in any browser:

```bash
# Clone the repo
git clone https://github.com/<your-username>/kamat-bakery.git

# Open in browser
open kamat_bakery/index.html
```

## Deploying to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)`
4. Your site will be live at `https://<your-username>.github.io/kamat-bakery`

## Customising

All content is in a single file — `index.html`. Things you may want to update:

- **Phone number** — search for `Contact us for phone number`
- **Address** — search for `Dharwad, Karnataka, India`
- **Opening hours** — search for `Mon – Sat`
- **Menu prices** — update `₹` values in the menu cards
- **Map coordinates** — currently set to `15.4634015, 75.0022747`

## Tech Stack

- HTML5, CSS3, Vanilla JavaScript
- [Google Fonts](https://fonts.google.com) — Playfair Display + Inter
- [Unsplash](https://unsplash.com) — placeholder images
- Google Maps Embed API
